## Project Theme

**Gamers Hub** is a gaming accessories e-commerce web application where customers can browse gaming products, register, log in, add products to cart or wishlist, and submit product requests. Admin users can manage products and accept or reject customer requests.

## Technologies Used

- React.js
- React Router DOM
- Axios
- CSS /Tailwind
- React Hooks
- LocalStorage
- Node.js
- Express.js
- MySQL
- JWT Authentication
- bcryptjs
- dotenv
- cors
- nodemon

## Main Features

- User registration and login
- Admin login and dashboard
- Product listing and product details
- Cart management
- Wishlist management
- Product request workflow
- Admin product management
- Admin request accept / reject
- User request status display
- Frontend and backend validation
- Database integration
- Error handling and success messages
